# KRITA-SakugaKR

A Krita docker plugin for 2D Animation, with features : <br>
<ul>
  <li>Camera Frame Stamp</li>
  <li>Auto Create Animation Layers and Import Animation Keyframes</li>
</ul>
<h2>How to Install</h2>
<ul>
  <li>Download the <a href="https://github.com/spikysaurus/KRITA-SakugaKR/archive/refs/heads/main.zip">.zip</a></li>
  <li>In Krita, go to “Tools” >> “Scripts” >> “Import Python Plugin from File…”>> Find and select main.zip</li>
  <li>To use the plugin, go to "Settings" >> "Dockers" >> find "SakugaKR"</li>
</ul>
